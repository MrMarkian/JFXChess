JFXChess 

A chess program written to further skills in Java FX & Java in general.

Supported Features
 - [x] FEN board setup
 - [ ] Multiplayer support & game viewing
 - [ ] Full PNG support
 - [ ] 3D models on 3d view of chess board.
 - [ ] Recursive AI engine(currently only 1 level deep)
 - [x] Chess Clocks
